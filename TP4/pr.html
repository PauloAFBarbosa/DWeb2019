Abstract:

Neste trabalho foi necessário dividir o xml dos Arqueossítio em xml's individuais e depois criar uma stylesheet para transformar os mesmos em html.

Por fim foi necessário criar um servidor em nodejs capaz de devolver o xml do Arqueossítio selecionado e a respetiva stylesheet para o transformar.

Deliverables:

  • Xml do PR.
  • Stylesheet responsável por transformar o PR em html.
  • Html do ficheiro PR.
  • Xml dos Arqueossítios.
  • Stylesheet responsável por transformar cada Arqueossítio em html.
  • Servidor em Nodejs.
  • Stylesheet responsável por transformar o ficheiro xml dos Arqueossítios em vários ficheiros individuais.
  • Pasta com os Arqueossítios individuais.
