Abstract:

Neste trabalho fez-se um website utilizando a framework js express.

Neste website é possivel adicionar/remover e editar registos de música. Para editar os mesmos é necessário clicar para ver e só depois editar os dados.

  • Stylesheet responsável por transformar o PR em html.
  • Html do ficheiro PR.
  • Ficheiro responsável por guardar as músicas.
  • Template em pug para os erros.
  • Template em pug para a página principal.
  • Template em pug do layout do site.
  • Ficheiro de rotas do index (index.js).
  • Ficheiro css usado no site.
  • Ficheiro com a função javascript responsável por fazer o pedido de delete ao servidor.
  • Ficheiro com a função javascript responsável por fazer o pedido de put ao servidor.
  • Ficheiro com a função javascript responsável por fazer o pedido de get ao servidor. É utilizado para mostrar um registo e editar o mesmo.
