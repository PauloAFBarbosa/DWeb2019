Abstract:

Neste trabalho fez-se um website utilizando a framework js express, onde o website guarda toda a sua informação numa base de dados utilizando o mongodb.

Neste website é possivel adicionar/remover e editar registos de filmes. Para editar os mesmos é necessário clicar para ver/editar e é possível também adicionar novos generos ao filme e novos elementos ao cast.
